# Basic Webserver

## Overview
A basic webserver implemented using Python and socket programming.

## Files
- 'webserver.py' - web server implementation
- 'test.html' - sample html page to confirm server functions

## How to Run
1. Clone this repository
2. start server:
```bash
    python webserver.py
```

3. open http://localhost:8080/test.html to see sample page

## Notes
- Server listens to port 8080 be default. Can change this by editing the port number of 'start_server()' in 'webserver.py'.
